What is React?

Components

Components are independent and reusable building blocks of web application

They allow us to split complex UI into small, isolated pieces

React components are basically just JS functions. They accept some inputs (called "props") and return React elements describing what should be displayed on the screen

Conponents can either be JS functions or classes

Funtion Component 

                  function Name() {
                    return 
Hi, my name is Joe!
;
                  }
Class Component 

                  class Person extends React.Component {
                    render() {
                      return 
Hi again from Joe!
;
                    }
                  }

We can use components inside other components

Props

  • Arguments passed into React components
  • Passed like HTML attributes
  • Props are immutable
  • In components they can be accessed via this.props

            const myelement = <Car brand="Ford" />; 
            
            class Car extends React.Component {
              render() {
                return 
I am a {this.props.brand}!
;
              }
            }

State

  • Property of React class component
  • Holds internal state of component
  • Accessed via this.state
  • When state changes component re-renders

            class Counter extends React.Component {
              state = {
                value: 0
              };
            }

Changing the state

To change a value in the state object we use this.setState() method


            class Counter extends React.Component {
              state = {

            handleIncrement= () => {

                 value: state.value + 1
